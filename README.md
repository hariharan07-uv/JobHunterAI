# 🚀 JobHunter AI

Automated Embedded Systems, Firmware, Semiconductor, and Electronics Job Aggregator with Telegram Notifications.

JobHunter AI continuously scans multiple company career portals, filters relevant ECE/Embedded/VLSI opportunities, stores them in CSV format, and sends real-time Telegram alerts.

---

# 📌 Why This Project Exists

Most job portals contain thousands of unrelated openings.

As an Electronics and Communication Engineering (ECE) student, finding relevant jobs such as:

* Embedded Software Engineer
* Firmware Engineer
* Linux Kernel Developer
* RTL Design Engineer
* FPGA Engineer
* ASIC Verification Engineer
* VLSI Engineer
* Hardware Engineer

takes a lot of manual effort.

JobHunter AI automates this process.

---

# ⚙️ System Architecture

```text
Company Career Portals

        │
        ▼
telegram_notify.py
```

# 📂 Repository Structure

## Core Files

### aggregator.py

Main engine of the project.

Responsibilities:

* Calls all company scrapers
* Filters jobs
* Removes unwanted roles
* Stores jobs in memory
* Generates all_jobs.csv

Output:

```text
all_jobs.csv
```

---

### telegram_notify.py

Responsible for Telegram notifications.

Reads:

```text
all_jobs.csv
```

Sends:

```text
Telegram Messages
```

Uses:

```text
TELEGRAM_BOT_TOKEN
TELEGRAM_CHAT_ID
```

from GitHub Secrets.

---

### requirements.txt

Python dependencies.

Installed automatically by GitHub Actions.

---

### all_jobs.csv

Final aggregated job database.

Generated by:

```text
aggregator.py
```

Contains:

* Company
* Role
* Location
* Application Link

---

### seen_jobs.txt

Stores previously processed jobs.

Purpose:

* Prevent duplicate alerts
* Future support for "new jobs only"

# 🤖 GitHub Actions Automation

Workflow File:

```text
.github/workflows/jobhunter.yml
```

Responsibilities:

1. Checkout repository
2. Install dependencies
3. Run aggregator.py
4. Generate all_jobs.csv
5. Send Telegram notification
6. Upload CSV artifact

Runs every:

```cron
0 */6 * * *
```

(Every 6 hours)

# 📱 Telegram Setup

Create a bot using:

```text
@BotFather
```

Obtain:

```text
BOT TOKEN
```

Find Chat ID:

```text
https://api.telegram.org/bot<TOKEN>/getUpdates
```

Add both as GitHub Secrets:

```text
TELEGRAM_BOT_TOKEN
TELEGRAM_CHAT_ID
```

# 🚀 Local Setup

Clone Repository

```bash
git clone https://github.com/hariharan07-uv/JobHunterAI.git
cd JobHunterAI
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Aggregator

```bash
python aggregator.py
```

Send Telegram Alert

```bash
python telegram_notify.py
```
